# RiskNova — AI Credit Risk Scoring System

A production-grade AI credit risk scoring system for Uganda's lending market.

## Features
- XGBoost + LightGBM ensemble models (AUC-ROC: 0.9454)
- SHAP explainability for every decision
- UGX-native monetary inputs
- AI Credit Counsellor powered by Claude API
- Live A/B testing framework
